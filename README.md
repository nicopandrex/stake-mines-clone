# Stake Mine's Clone
A low level clone of Stake's Mines gambling game. Originally created in 2024 for my Senior year CS class.

# Tech Stack
Written in Java using Processing 4 (necessary to run)

# How to Play
You start with $1000, select how much you want to bet each round, each tile you click your bet gets increasingly multiplied, keep going or cashout! If you hit a mine you lose it all
